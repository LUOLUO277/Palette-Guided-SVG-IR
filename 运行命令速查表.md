## 4. NVS 评测

说明：

- 如果仓库路径有空格，建议从无空格别名路径执行
- 当前我实际使用的是 `C:\tmp\svgir`

先准备无空格入口：

```powershell
if (-not (Test-Path -LiteralPath 'C:\tmp')) { New-Item -ItemType Directory -Path 'C:\tmp' | Out-Null }
if (Test-Path -LiteralPath 'C:\tmp\svgir') { Remove-Item -LiteralPath 'C:\tmp\svgir' -Force }
New-Item -ItemType Junction -Path 'C:\tmp\svgir' -Target 'D:\Grade Three Down\Experiment\SVG-IR' | Out-Null
cd C:\tmp\svgir
```

然后执行：

```powershell
. .\script\setup_native_windows.ps1 -SkipCondaCuda -SkipPipDeps -SkipBuild | Out-Null
$env:PYTHONIOENCODING='utf-8'
& "C:\Users\c2483\anaconda3\envs\svgir37p\python.exe" .\eval_nvs.py `
  --eval `
  -s C:\tmp\svgir\dataset\TensoIR\armadillo `
  -m C:\tmp\svgir\output\TensoIR\armadillo\render_relight_local_r4_i50000 `
  -c C:\tmp\svgir\output\TensoIR\armadillo\render_relight_local_r4_i50000\chkpnt50000.pth `
  -t render_relight `
  --skip_train
```

结果文件：

- 指标: [metric_test.txt](D:\Grade Three Down\Experiment\SVG-IR\output\TensoIR\armadillo\render_relight_local_r4_i50000\metric_test.txt)
- 导出: [test/ours_50000](D:\Grade Three Down\Experiment\SVG-IR\output\TensoIR\armadillo\render_relight_local_r4_i50000\test\ours_50000)
